# Productivity & Workflow

Morning/Evening Workday Toggle: Automate your "Start" and "End" of day routines.

Trigger Shortcut & Do Not Disturb: Sets Focus mode and executes specific macOS Shortcuts.

Numbers Daily Logger: Automated data entry for tracking daily metrics in Apple Numbers.

Restart Safari: Closes Safari and reopens the last active session.

# File & Folder Management

New Folder from Selection: Creates a new folder directly next to selected items in Finder.

AutoProcess Camera Folder: Automatically organizes or processes incoming images from camera imports.

Screenshot Uploader (Bash): A shell script to identify and upload your latest screenshots to a remote server or cloud.

Shrinkage: A utility script for file compression or footprint reduction.

# System & Hardware Utilities

Mount Local Network Volumes: Automatically maps network drives upon system startup.

Unmount MSI Monitor Driver: Auto-ejects the driver volume often mounted by MSI monitors.

Reset Touch Bar: Force-restarts the Touch Bar agent on MacBook Pros when frozen.

# Usage

Each snippet is identified by its unique reference ID. To use these scripts:

Open Script Editor on your Mac.

Copy the desired code block.

Save as a .scpt or .app for easy access.

# IMPORTANT

Some scripts may require Accessibility or Full Disk Access permissions under System Settings > Privacy & Security.
